About

About Me

Hello, my name is Katie Fehlberg. I recently graduated from the College of Design at Iowa State University. I have spent the last year immersing myself in the world of web development.

Through various courses and sources, I have practiced programming using HTML, CSS, and JavaScript. I have experimented with different design types and strategies, including fluid, adaptive, responsive, and mobile-first. I have explored using tools and libraries including Sass, Git, jQuery, and Modernizr. I have played with different effects and methods of implementation using CSS animations, transforms, and transitions, as well as SVG animations, and jQuery effects. I have observed the drawbacks and quirks of different browsers and have researched fallbacks for those that lack support for certain features.

In my spare time, I enjoy practicing yoga with my cat, Frisco; hiking, biking, and kayaking; playing board games with friends; eating chocolate in any form; Watching classic movies (especially those that feature Cary Grant); trying different microbrews; and using my hands to create anything from pottery to apple pancakes.
